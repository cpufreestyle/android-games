# Android 小游戏合集 🎮

收录多个 Android 小游戏项目，使用 Python 开发。

## 项目列表

### 1. 🎰 抽奖应用 (lottery-app)
- 语言：Python
- 功能：抽奖/抽签应用
- 文件：lottery.py, requirements.txt

### 2. 🔵 弹珠 3D 服务 (marble-3d-service)
- 语言：Python / 3D
- 功能：3D 弹珠游戏服务端
- 文件：详见 marble-3d-service 目录

### 3. 🐍 贪吃蛇游戏 (snake-game)
- 语言：Java (Android)
- 功能：经典贪吃蛇游戏
- 文件：SnakeGame.apk, 等

## 使用方法

每个游戏有独立的子目录，请查看对应目录中的 README.md。

### 运行 Python 项目
```bash
cd lottery-app
pip install -r requirements.txt
python lottery.py
```

### 安装 Android APK
将 SnakeGame.apk 传输到 Android 手机，启用"未知来源"后安装。
