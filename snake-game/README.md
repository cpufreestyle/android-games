# 🪱 贪吃蚯蚓 (Greedy Worm)

一款以真实蚯蚓为灵感的 Android 贪吃蛇游戏。

## 特色

- **圆形分段身体** — 蚯蚓不再是方块，而是圆形分段，越靠近尾部越细
- **蠕动摇摆动画** — 移动时身体自然左右摇摆，如真实蚯蚓般蠕动
- **头部眼睛** — 眼睛会跟随移动方向自动转向
- **环带纹理** — 第3~5节有深色环带（真实蚯蚓的繁殖带特征）
- **地下世界** — 泥土背景、草地表面、随机根系装饰
- **大地色系** — 肉粉、沙棕、土壤褐色的自然配色

## 安装说明

1. 下载 APK 文件到手机
2. 允许"安装未知应用"权限
3. 直接安装即可游玩

## 游戏操作

- **方向键 / 滑动屏幕** — 控制移动方向
- **加速按钮** — 加快移动速度

## 技术信息

- 平台：Android
- 语言：Java + XML
- 构建工具：Gradle
